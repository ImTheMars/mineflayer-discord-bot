
# Mineflayer Discord Bot

Control your Minecraft player using Mineflayer & discord.js




## Features

- Automated Mining
- Pathfinder
- Message
- Statistics
- PVP




## Environment Variables

To run this project, you will need to add the following to your config file

`token`
`username`
`password`
`host`
`port`
`version`
`auth`
